# Vector Search
The goal here is to implement different ANN algos and techniques.

For py directory, the goal is to not use specialized external libraries (ie. via pip install) and make them as simple as possible to read (for the purpose of learning), thus it will be slow.

## HNSW
Used the following to implement hnsw.
- Git: https://github.com/nmslib/hnswlib/tree/master
- Paper: https://arxiv.org/pdf/1603.09320

## PQ
Used the following to implement PQ.
- KMeans: https://en.wikipedia.org/wiki/K-means_clustering
- KMeans++: https://en.wikipedia.org/wiki/K-means%2B%2B
- PQ Paper: https://inria.hal.science/inria-00514462v2/document
